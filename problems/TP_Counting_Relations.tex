\begin{problem}

%% type: short-answer
%% title: Counting Relations

How many relations are there on a set of size $n$ when:
\bparts

\ppart
$n=1$?
\begin{solution}
2.
\end{solution}
\ppart
$n=2$?
\begin{solution}
16. 

Four pairs (i.e. AA, AB, BA, BB), each of which can be present or absent, gives a total of $2^4=16$ possible binary relations.
\end{solution}

\eparts


\end{problem}
