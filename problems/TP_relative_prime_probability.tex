\begin{problem}
Suppose $p$ and $q$ are distinct 400 digit primes, and let $n = pq$.
Which of the following expressions is the smallest guaranteed upper
bound on the probability that a random integer in the interval
$\Zintv[1,n]$ is not relatively prime to $n$?   (You may
assume the fact that there is a prime number in the interval
$\Zintvoc[m,2m]$ for all positive integers $m$.)


\begin{itemize}
\item $10^{-400}$
\item $10^{-800}$
\item $2 \cdot 10^{-400}$
\item $10^{-401}$
\item $10^{-399}$
\end{itemize}

\begin{solution}
$2 \cdot 10^{-400}$.

The number of integers in $\Zint[1,n]$ that are not relatively prime
to $n$ is $n-\phi(n) = pq - (p-1)(q-1) = p + q - 1$, so the
probability of randomly choosing one of these is
\[
(p + q - 1)/n = 1/q + 1/p - 1/pq
\]
Since any 400 digit number is $\geq 10^{399}$.
